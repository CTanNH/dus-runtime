# Benchmark the knowledge workspace against a box-based baseline

## Problem

The repo has a strong thesis, but it still needs one sharp empirical proof: that DUS helps with a real task better than a naive list/card UI.

## Goal

Define one narrow benchmark for the AI knowledge workspace and compare DUS against a conventional baseline.

## Candidate Tasks

- trace evidence behind a claim
- identify contradictions
- spot low-confidence regions in an answer

## Deliverables

- a baseline box-stack version of the same knowledge workspace
- at least one repeatable task scenario
- measurements or structured observations on completion time / error rate / user comprehension
- a short write-up in the repo documenting results and caveats

## Exit Criteria

- the project can point to a concrete, non-aesthetic advantage
- the benchmark is reproducible enough to guide future design decisions
- the result informs roadmap priority instead of remaining a vague aspiration
