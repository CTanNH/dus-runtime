# DUS v0.1.0

Initial public release of DUS as a research-grade runtime prototype.

## Included

- headless semantic layout runtime
- scaffold + hybrid solver split
- WebGPU renderer adapter with `plain`, `field`, and `debug` surfaces
- DOM host bridge and inspector
- AI knowledge workspace demo
- MSDF text and raster image pipeline
- manifesto, architecture, and roadmap docs

## Not Included

- stable production API
- accessibility-complete host integration
- large-scene GPU solver path
- benchmark proof against a conventional box-based interface

## Release Positioning

This release is not a finished framework. It is the first public articulation of DUS as:

> a loss-driven semantic layout runtime for AI-native interfaces
